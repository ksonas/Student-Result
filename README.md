# Student Result Management System
A simple Python program to manage student results using 10th-grade subjects. This system helps in entering marks, calculating grades, and storing records in a CSV file.

## Features
- Add student details
- Enter marks for Maths, Science, Social Science, English and Hindi
- Calculate total and average
- Generate grade automatically
- Store data in CSV file
- View all student records

### Setup
1. Install Python and MySQL
(Only Python is required for this project)
2. Open the project folder
3. Run the program: `python main.py`
